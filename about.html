Portfolio A memory matching game in the theme of classic PlayStation title Metal Gear Solid. I wanted to give the game that authentic 1998 feel by utilizing assets such as artwork from the game's character designer, a custom font package and an image scanned directly from magazine ad. Additionally the color pallet was chosen by lifting colors directly from in game screenshots in order to stay true to the games original visuals themes.
